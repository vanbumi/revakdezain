### Catatan Penting buat Kamu (Untuk Template HTML di Renderer):
Agar Cline bisa mengeksekusi ini dengan hasil yang tampak "mahal", pastikan template HTML yang dibaca Puppeteer memiliki struktur CSS berlapis (*layering*) seperti ini:

 1. **Gunakan Font Bagus:** Panggil font komersial/modern di tag <head> template HTML, misal:
   <link href="https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@400;700;800&display=swap" rel="stylesheet">

 2. **Buat Background Berlapis (Mesh Gradient + Efek Gambar):**
   Di dalam komponen background Tailwind-mu, gabungkan warna gradasi dari DeepSeek dengan efek gambar (setelah Pexels aktif nanti), contoh struktur dasarnya:

   ```html
   
   ```
<div
style="background: linear-gradient(135deg, ${data.visual.gradientFrom}, ${data.visual.gradientTo});"
class="relative w-full h-full text-white font-['Plus_Jakarta_Sans'] p-16 flex flex-col justify-between overflow-hidden"
> 
<!-- Efek Ornamen Watermark Grid Grafis bawaan template (Manusia Banget) -->
<div class="absolute inset-0 opacity-10 bg-[linear-gradient(to_right,#808080_1px,transparent_1px),linear-gradient(to_bottom,#808080_1px,transparent_1px)] bg-[size:40px_40px]"></div>
<!-- Lingkaran Cahaya Blur (Glow Effect) di pojok untuk estetika -->
<div style="background: ${data.visual.warnaAksenTeks};" class="absolute -top-20 -left-20 w-80 h-80 rounded-full blur-[150px] opacity-30"></div>
<!-- KONTEN IKLAN (Headline, Offer, CTA) tinggal disusun rapi menggunakan flexbox -->
</div>

### Perintah Tambahan untuk Cline (Kunci Grafis Estetik):
> "Cline, saat kamu menyusun template HTML/Tailwind di dalam fungsi renderPuppeteer, kamu **wajib* menambahkan sentuhan desain grafis profesional (Aesthetics Multi-Layering) agar latar belakangnya tidak polos. Terapkan elemen berikut langsung di dalam kode HTML-nya:"*

>    bg-[linear-gradient(to_right,#ffffff08_1px,transparent_1px),linear-gradient(to_bottom,#ffffff08_1px,transparent_1px)] bg-[size:32px_32px]
